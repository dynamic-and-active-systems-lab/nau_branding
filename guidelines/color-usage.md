# NAU Color Usage Guidelines

Source: [in.nau.edu/visual-identity-guide/color/](https://in.nau.edu/visual-identity-guide/color/)

## Primary Palette

| Color | Hex | Pantone | Role |
|-------|-----|---------|------|
| NAU True Blue | `#002454` | 282 C | Dominant brand color — must appear in all communications |
| NAU Gold | `#FAC01A` | 3514 C | Accent and logo mountain element |

**True Blue is NAU's signature color.** It must appear as the dominant color in all university communications. When in doubt, default to True Blue.

### Secondary Color Rules

- **80/20 balance**: 80% primary (True Blue + Gold), 20% secondary.
- Secondary colors add vibrancy and visual interest — they must never dominate True Blue and Gold.
- **Do not apply secondary colors to body text.** Any text over a colored background must pass ADA contrast compliance.
- **Sky blue** must be used very sparingly. The visual impression of the NAU brand must remain dark blue.

## Accessibility

All text over any background color must meet WCAG AA contrast ratios:
- Normal text (under 18pt): minimum 4.5:1 contrast ratio
- Large text (18pt+ or 14pt+ bold): minimum 3:1 contrast ratio

**Known failure:** The two-color logo with gold type **must not be used on the web** — it fails accessibility contrast. Use True Blue type version on web.

Use NAU's contrast checker: [color.a11y.com](https://color.a11y.com) or the WCAG checker linked from the color guide.

## Print vs. Digital

Always use the exact values for each medium:
- **Print**: Use Pantone (spot) or CMYK values
- **Digital/Web**: Use hex or RGB values
- Do not substitute or approximate — even small deviations break brand consistency across print runs

## Do Not

- Introduce colors outside the approved palette
- Use tints or screens of brand colors (e.g., 50% True Blue)
- Use secondary colors as primary backgrounds
- Place the gold logo mountain over a gold background
