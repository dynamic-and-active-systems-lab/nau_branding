# NAU Logo Usage Guidelines

Source: [in.nau.edu/visual-identity-guide/university-marks/](https://in.nau.edu/visual-identity-guide/university-marks/)

> **Every piece of NAU communication must include the NAU logo.**

## Logo Types (in order of preference)

| Type | When to Use |
|------|------------|
| Primary (stacked) | Default for all uses |
| Two-Line Horizontal | Not enough vertical space for primary |
| One-Line Horizontal | Space is even more horizontal |
| Centered | Only in rare cases where nothing else fits |
| Wordmark (stacked or horizontal) | Imprint area < 1" wide, web footer, postcard return address |
| Ligature only | Internal materials or audiences already familiar with NAU — wordmark must appear subordinately |
| Heritage/Ancillary | Formal communications conveying institutional prestige only |
| University Seal | Office of the President and ABOR-sponsored applications only |

## Preferred Coloration

- **Mountain graphic**: NAU Gold (Pantone 3514)
- **All other elements (wordmark)**: NAU True Blue (Pantone 282)

## Color Variations

### On Light Backgrounds (Positive)

| Variation | When to Use |
|-----------|------------|
| Two-Color, True Blue type *(preferred)* | Default — web, stationery, most applications |
| True Blue (all blue) | Merchandise, logo on Gold background |
| Black | Black-and-white print only |

### On Dark Backgrounds (Reversed)

| Variation | When to Use |
|-----------|------------|
| Reversed Gold | Over True Blue background or other dark colors with sufficient contrast |
| Reversed White | One-color print jobs, or backgrounds that vibrate against gold |

**Do not use** the gold mountain graphic when nothing else in the logo is gold.

## Web-Specific Rules

- **Never** use the Two-Color with Gold Type variation on the web — it fails ADA contrast.
- Always use Two-Color with True Blue Type on web.

## Minimum Sizes

| Logo | Print Minimum | Screen Minimum |
|------|--------------|----------------|
| Institutional (most colorations) | 0.75 in | 185 px |
| Institutional — Two-Color Gold Type | 1.0 in | 246 px |
| Unit Logo — Primary | 1.375 in | 325 px |
| Unit Logo — Horizontal | 1.125 in | 276 px |
| Heritage Logo | 1.5 in | — |
| University Seal | 1.75 in min / 3.5 in max | — |

## Clear Space

Maintain a minimum clear space equal to **the height of the ligature "A"** on all four sides of the logo, and between the logo and any other element (text, graphic, image, page edge).

## Cross-Branding (Partner Logos)

- NAU logo must appear **left** or in the **most dominant position**
- Use the Two-Color, True Blue type configuration
- Partner logo text must not be larger than the NAU wordmark
- Partner logo must not dominate the NAU logo
- Logos should not be placed directly next to each other — maintain NAU clear space between them
- **Envelopes**: partner logos may NOT appear on NAU envelopes

## Social Media

| Account Type | Required Logo |
|-------------|--------------|
| Prospective-audience-facing | Stacked primary institutional logo |
| NAU Social / university-level | Gold type stacked logo permitted |
| Department (internal audience) | Unit logo or ligature permitted |
| Ligature alone | Only when "Northern Arizona University" is spelled out nearby |

## Unacceptable Treatments

**Never do any of the following:**

- Use the gold type version at < 1" wide in print, or at any size on the web
- Crop any logo element
- Distort or stretch the logo
- Rearrange logo elements (even in correct proportions)
- Apply a vignette effect
- Delete any element
- Tint, screen back, or apply opacity to the logo
- Rotate the logo
- Place the logo over a busy portion of a photo
- Place the gold mountain over a gold background
- Reproduce in any color combination not listed above
- Apply gold type to unit logos
- Combine the NAU logo with another logo to create a single merged image

### File Format Guide

| Format | Use When |
|--------|---------|
| EPS | Any print project — infinitely scalable vector |
| PNG | Web and digital only — use at 100% of saved size |

## Louie the Lumberjack (Mascot)

Use is permitted by university departments provided:
- No alterations of any kind
- No accompanying verbiage, text, or design elements
- Adheres to University Trademark Licensing Policy
- Minimum clear space maintained (same as logo guidelines)
