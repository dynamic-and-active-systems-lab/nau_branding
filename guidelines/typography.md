# NAU Typography Guidelines

Source: [in.nau.edu/visual-identity-guide/typography/](https://in.nau.edu/visual-identity-guide/typography/)

## Font Decision Tree

| Context | Primary Font | Fallback |
|---------|-------------|---------|
| Web pages | Acumin Pro | Arial |
| Print marketing | Univers | Arial |
| Email / correspondence | Calibri | Arial |
| Formal documents | Adobe Caslon Pro | Times New Roman |
| Logo | Bank Gothic (do not use elsewhere) | — |

## Primary Institutional Font: Univers

Univers (regular weight family, **not** condensed) is the university-wide institutional sans-serif. Use it for all university-wide marketing and communications when available.

**Licensed** — purchase at myfonts.com or fonts.com.

| Weight Name | CSS Weight |
|-------------|-----------|
| Univers 75 Black | 900 |
| Univers 65 Bold | 700 |
| Univers 65 Bold Oblique | 700 italic |
| Univers 55 Roman | 400 |
| Univers 55 Oblique | 400 italic |
| Univers 45 Light | 300 |

*Condensed variants (Univers 55 Condensed, 47 Condensed Light) are available but reserved for specific layout needs.*

**Do not use Univers on the web.** Use Acumin Pro instead.

## Web Font: Acumin Pro

Always use Acumin Pro for NAU web pages. Only these three variants are approved:

1. **Acumin Pro** — standard weight, body copy and headings
2. **Acumin Pro Condensed** — space-constrained layouts
3. **Acumin Pro Extra Condensed** — extreme space constraints only

Using different weights creates clear hierarchy between headlines and body copy.

## Supplementary Fonts

**Calibri** — for correspondence and email when Univers is unavailable. Default for email signatures.

**Adobe Caslon Pro** — serif font, use sparingly for formal documents with sophisticated audiences. Fallback: Times New Roman.

## Print Specifications

### Business Card (2" × 3.5")
- Name: Univers 65 Bold, 7pt / 11pt leading, Pantone 281
- Title: Univers 45 Light, 7pt / 11pt leading, Pantone 281
- Contact info: Univers 45 Light, 7pt / 11pt leading, Pantone 281

### Letterhead
- Unit name: Univers 65 Bold, 7.5pt / 11pt leading, Pantone 281
- Contact info: Univers 45 Light, 7.5pt / 11pt leading, Pantone 281
- Body copy (word processing): Calibri Light, 10pt, black

### Email Signature
- Name and title: Calibri Bold, 10pt, NAU True Blue or black
- All other lines: Calibri, 10pt, NAU True Blue or black
- Separate components with vertical rules ( | ) with two spaces on each side
