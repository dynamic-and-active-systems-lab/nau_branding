# NAU Logo Files

Logo files are **not stored in this repository** because they are trademark-protected assets that require authorized distribution through official NAU channels.

## Where to Get Logo Files

| Need | Source |
|------|--------|
| Download approved logo files | [nau.edu/digitalassets/widen-collective](https://nau.edu/digitalassets/widen-collective) |
| Request a specific logo | [nau.edu/logorequest](https://nau.edu/logorequest) |
| Custom unit logo creation | Contact University Marketing |
| Seal usage approval | Contact University Marketing |

## File Formats Available

| Format | Use Case |
|--------|---------|
| `.eps` | All print projects — vector, infinitely scalable |
| `.png` | Web and digital — use at exactly the saved resolution |
| `.jpg` | Photography composites — avoid for logos when possible |
| `.wmf` | Windows/Office applications |

## File Naming Convention

`NAU_Description_Configuration_Coloration.eps`

**Configuration codes:**
- `primary` — Stacked (primary)
- `horiz_2line` — 2-line horizontal
- `horiz_1line` — 1-line horizontal
- `ctr` — Centered
- `wordmark` — Wordmark only

**Coloration codes:**
- `281_3514` — Two-color: True Blue type, Gold mountain *(preferred)*
- `3514_281` — Two-color: Gold type, Gold mountain *(print only, not web)*
- `281` — All True Blue
- `K` — All black
- `Rev-3514` — Reversed: Gold and white type with Gold mountain
- `Rev-3514-wht` — Reversed: white type with Gold mountain
- `Rev-wht` — Reversed: all white

## Important Restrictions

- NAU logos are trademark-protected. All merchandise use requires approval from the Office of Trademarks & Licensing (licensing@nau.edu / 928-523-5404).
- Do not store, distribute, or embed logo files in public repositories.
- For Claude design use: reference the logo specifications in `../guidelines/logo-usage.md` and link to Widen Collective for actual files.
